# pullFromAll
Bash script to automatically pull from repositories in a directory

## Example Directory Hierarchy:

```
 MyRepositories
 |
 |_ pullFromAll.sh
 |_ repo1/
 |_ repo2/
 |_ ...
```

## How to Use:
   Copy the file, make it executable with ``chmod +x pullFromAll.sh``, call with ``bash pullFromAll.sh``.
